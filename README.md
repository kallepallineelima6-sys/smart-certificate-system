# Smart Certificate System

AI-powered certificate management system using OCR and trust classification.

---

## 🚀 Features

- OCR-based certificate data extraction
- Trust Score System:
  - 🟢 Verified
  - 🟡 Semi Verified
  - 🔴 Unverified
- Government document detection (PAN, Aadhaar)
- Expiry alerts system
- Emergency access sharing
- Smart categorization (Education, Govt, Skills)

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- OCR: Tesseract OCR

---

## 📂 Project Structure

- index.html → Home page
- login.html → Login page
- dashboard.html → Main dashboard
- server.js → Backend server
- script.js → Frontend logic
- styles.css → UI design

---

## ▶️ How to Run

1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Open in browser:
   http://localhost:3000

---

## 💡 Use Case

This system helps users:
- Store certificates digitally
- Automatically extract data
- Verify authenticity using AI logic
- Get alerts for expiry
- Share documents securely
